# Supported networks

Sunodo deployment supports all networks officially supported by the Cartesi rollups framework:

-   Ethereum
    -   Mainnet: `mainnet`
    -   Testnet: `sepolia`
-   Arbitrum
    -   Mainnet: `arbitrum`
    -   Testnet: `arbitrum-goerli`
-   Optimism
    -   Mainnet: `optimism`
    -   Testnet: `optimism-goerli`
